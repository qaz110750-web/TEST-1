실시간 환율 계산기

한국수출입은행 환율 데이터를 사용합니다. API KEY는 브라우저에만 저장됩니다.

API KEY와 날짜를 입력한 뒤 환율을 불러오세요.
한국수출입은행 API는 영업일 기준 데이터가 제공될 수 있습니다. 선택 날짜에 데이터가 없으면 가까운 이전 영업일로 다시 조회해보세요. JPY는 API에서 100엔 기준으로 제공되는 경우가 많아, 계산 시 1엔 기준으로 자동 환산합니다.
